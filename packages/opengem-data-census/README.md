# opengem-data-census

🧪 alpha

US Census Bureau adapter. Free; optional API key for higher rate limits.

## Catalog (IOC)

| OPENGEM SeriesId | Census endpoint |
|---|---|
| `US.CENSUS.M3.inventories_total.M` | M3 inventories total |
| `US.CENSUS.M3.shipments_total.M`   | M3 shipments total |
| `US.CENSUS.M3.new_orders_total.M`  | M3 new orders total |
| `US.CENSUS.MRTS.retail_sales.M`    | MRTS total retail sales |
